SophtMade — Debug Tool

Open this file from inside your sophtmade folder. Check each section below.

1. products.js loaded?

2. PRODUCTS array

3. Image paths — click to test each one

4. Console errors (check F12 → Console yourself for red errors)

Open F12 → Console and look for red errors. The most common ones are:

• "PRODUCTS is not defined" → products.js not loading
• "Cannot read properties of undefined" → a product has a missing field
• "Unexpected token" → syntax error in products.js (check for missing commas)
• 404 errors on .js files → wrong folder structure
